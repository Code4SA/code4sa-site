Elections API

The Code4SA Elections API provides a machine-readable dataset sourced from the Electoral Commission of South Africa. It allows you to build a news application, a web page, or any other type of software that needs elections data.

Medicines API

Retail medicine prices are regulated in South Africa through the Single Exit Price Mechanism. Prices are published on a regular basis at www.mpr.gov.za in an unwieldy excel spreadsheet making it inaccessible to casual users. This projects builds an API and basic user interface to query this database.
